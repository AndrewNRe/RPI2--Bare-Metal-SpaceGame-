設定:
ハードウェア:
Peter Lemonの図表が最高 -　https://github.com/PeterLemon/RaspberryPi/blob/master/Input/SNES/Controller/SNESController.png
ソフトウェア:
Micro SD cardで4個ファイルが必要です。
2個がRaspberry PiのGithubでダウンロードされます。(https://github.com/raspberrypi/firmware/tree/master/boot)
終わりのファイル設定が
bootcode.bin (Raspbery PiのGithubで)
config.txt (ダウンロードのフォルダ)
start.elf (Raspbery PiのGithubで)
kernel7.img (ダウンロードのフォルダ、これはゲームです!)

遊べない場合があれば私に知らせてください!

遊び方:
コントロールでAとBを押すれば「Mode」が変わられる。
プラットフォームからプラットフォーム飛んで着陸したら、得点が上に行こう!
落ちたら、得点を消させる。
